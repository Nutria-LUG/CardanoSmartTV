Detailed Description

This television driver is used for debug purpose. It take an other driver, like proxy pattern, to perform the core operations. This is used to make assertion on pre conditions before each operations and post condition after execution of each methods. See TelevisionAdapter documentation to know the Contracts to be checked.
NOTE: The proxied object is taken as unique_ptr and the ownership is moved to this object through std::move function.
